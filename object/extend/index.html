#extend API documentation

Syntax

core.extend(source)(target);
Parameter Type Description
source Object Source object
target Object Target object

Usage

See Object.assign. Extend object: source → target.

core.extend({
    a: 0,
    b: null,
    c: 2,
})({
    b: 1,
    d: 3,
}); /* {
    a: 0,
    b: 1,
    c: 2,
    d: 3,
} */
