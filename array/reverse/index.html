#reverse API documentation

Syntax

core.reverse(array);
Parameter Type Description
array any[] Array to reverse

Usage

See Array.prototype.reverse. Reverse array.

core.reverse([ 0, 1, 2, 3 ]); /* [ 3, 2, 1, 0 ] */
